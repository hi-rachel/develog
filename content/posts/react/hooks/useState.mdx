# useState 완벽 가이드

React의 useState는 함수형 컴포넌트에서 상태를 관리할 수 있게 해주는 Hook입니다.

## 기본 사용법

```jsx
const [state, setState] = useState(initialValue);
```

## 언제 사용해야 할까?

1. 컴포넌트 내부에서 변경될 수 있는 값을 관리할 때
2. 리렌더링이 필요한 데이터를 다룰 때

## 주의사항

1. 컴포넌트의 최상위 레벨에서만 호출
2. 조건문 내부에서 사용하지 않기
